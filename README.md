# 3장

이 장에서는 컴포넌트를 만들어서 내보내고 불러오는 방법과 props , state를 사용하는 방법을 배웁니다. 

props와 state 둘 다 컴포넌트에서 사용하거나 렌더링할 데이터들을담고 있으므로 비슷해 보일수 도 있지만 역할은 매우 다릅니다.

 props는 부모 컴포넌트가 설정하고, state는 컴포넌트 자체적으로 지닌 값으로 컴포넌트 내부에서 값을 업데이트합니다.
